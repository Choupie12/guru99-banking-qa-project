# Test Cases

This folder contains the test cases created and executed during the testing of the Guru99 Banking Application.

Testing activities were performed across different testing levels:

Unit Testing 
Integration 
System Testing

Each file contains detailed test scenarios including:

Test steps
Expected results
Actual results
Execution status (Pass/Fail)

The objective was to validate application functionality and support defect identification throughout the testing lifecycle.

The test cases cover : 
- New Customer : Creating customer profiles
- Edit Customer : Modifying customer information
- New Account : Opening bank accounts
- Edit Account : Updating account details
- Delete Account : Account closure functionality
- Delete Customer : Removing customer records
- Mini Statement : Quick transaction summaries
- Customized Statement : Detailed account reports
- Deposit Statement : Deposit money into an account
- Withdrawal Statement : withdraw money from an account
- Balance Enquiry : View balance of an account
- Fund Transfer : Transfer funds from his an account to any destination account
- Login/Logout as manager
- Login/Logout as customer
- Change Password
