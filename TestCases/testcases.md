# Test Cases

This folder contains manual test cases created for the Guru99 Banking Application testing project.

The test cases cover:
- Login/Logout functionality
- Fund transfer
- Customer creation & deleting
- Deposit and withdrawal
- Balance enquiry

File included:
- TestCase_v1.xlsx
- TestCase_v2.xlsx
- IntegrationTesting_v1.xlsx
- IntegrationTesting_v2.xlsx
- SystemTesting_v3.xlsx
