# Guru99 Banking Application - Manual Testing Project

## Project Overview

This repository contains a complete manual testing project conducted on the Guru99 Banking Application.

The project covers different levels of software testing and defect management activities performed throughout multiple versions of the application.

## Project Objectives

The main objectives of this project were to:

- Design and execute test cases
- Validate application functionalities
- Identify and document defects
- Perform retesting activities on updated versions
- Produce testing documentation and reports

## Testing Scope

The following testing levels were covered:

- Unit Testing
- Integration Testing
- System Testing

## Project Structure

### TestPlan
Contains the test planning documentation.

### TestCases
Contains the test cases executed during:
- Unit Testing 
- Integration Testing 
- System Testing

### BugReports
Contains defects identified during testing activities and their tracking across application versions.

### Screenshots
Contains screenshots collected during test execution and defect reporting.

### TestSummary
Contains the overall testing results and conclusions.
