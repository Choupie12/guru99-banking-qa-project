# Test Summary Report

## Project Information

**Project Name:** Guru99 Banking Application

**Testing Types:**

* Unit Testing
* Integration Testing
* System Testing

---

## Project Objective

The objective of this project was to validate the functionality, reliability and quality of the Guru99 Banking Application through structured manual testing activities.

The project included test design, test execution, defect identification, defect reporting and retesting activities across multiple application versions.

## Conclusion

A total of 279 test cases were executed across multiple testing levels, resulting in the identification and documentation of 179 defects throughout four application versions.

The testing activities contributed to validating application functionality, identifying software defects and verifying corrective actions through successive testing cycles.

This project provided hands-on experience in manual testing, defect management, test documentation and software quality assurance practices.
